Items

Here, you will find information about all new items and their required components.

Note 1: some of the upgrade ingredients you will see below are brand new items added by this mod to the game. These new components are rare and hard-to-attain. Think well before deciding to use any of them, because there is no guarantee that you will find them soon again during the game.

Note 2: some of the upgrade ingredients are categorized as gems and trinkets. If you have an optional component "Simplified economy" installed, the blacksmiths won't require these anymore to craft the item.
